## 📋 Sobre o projeto

Site de portfólio de página única (single-page) com design moderno, tipografia grande e layout em grid responsivo. Desenvolvido do zero sem frameworks ou bibliotecas externas.

**Seções:**
- 👤 Introdução — apresentação e título profissional
- 💼 Experiência — histórico profissional com linha do tempo
- 🎓 Formação — graduação, cursos intensivos e idiomas
- 📞 Contato — email, telefone e LinkedIn

---

## ✨ Destaques técnicos

- ✅ CSS modular com `@import` por componente
- ✅ Layout responsivo com CSS Grid e Flexbox
- ✅ Design system com variáveis de tipografia e cores
- ✅ CSS minificado para produção (`style.min.css`)
- ✅ Sem dependências externas — HTML e CSS puro
- ✅ Publicado via GitHub Pages
